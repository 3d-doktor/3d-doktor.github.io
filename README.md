# Runner Rush (GitHub Pages)

Ein kleines Browser-Spiel im Stil von **Subway Surfers**, gebaut mit HTML/CSS/JavaScript und ohne externe Abhängigkeiten.

## Features
- 3 Spuren mit Spurwechsel
- Springen und Sliden
- Zufällige Hindernisse + Münzen
- Punkte, Highscore (localStorage) und ansteigende Geschwindigkeit
- Touch- und Tastatur-Steuerung

## Steuerung
- **← / →**: Spur wechseln
- **↑**: Springen
- **↓**: Sliden
- **R**: Neustart

## Lokal starten
Öffne einfach `index.html` im Browser.

## GitHub Pages
Für GitHub Pages reicht es, den Inhalt dieses Repos auf den `main`-Branch zu pushen.
